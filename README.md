# 安防设备清单统计工具

一个纯前端的安防设备清单管理工具，无需安装，打开即用。

## 功能

- 支持 Excel 文件导入（.xlsx / .xls）
- 设备清单统计与汇总
- 数据导出为 Excel

## 使用方式

直接访问：[在线使用](https://你的用户名.github.io/security-equipment-tool/)

或下载 `index.html` 后，双击在浏览器中打开即可。

## 技术栈

- 纯 HTML / CSS / JavaScript
- [SheetJS](https://sheetjs.com/) 处理 Excel 读写
